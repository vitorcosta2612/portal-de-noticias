# Portal de Notícias

Um portal de notícias moderno e responsivo desenvolvido como um clone de grandes sites de notícias e tecnologia. O projeto foi criado com foco em reproduzir a estrutura visual, organização de conteúdo e experiência de navegação encontrada em portais profissionais.

## Sobre o projeto

Este projeto simula a interface de um portal de notícias real, com seções organizadas, destaques principais, categorias e cards de notícias.

A proposta foi reproduzir elementos comuns encontrados em grandes sites jornalísticos, como:

* Manchetes principais em destaque
* Notícias secundárias organizadas em grid
* Seções divididas por tema
* Área de notícias mais lidas
* Navegação lateral
* Campo de busca
* Cards com imagem, categoria e descrição
* Estrutura limpa e moderna

O site utiliza notícias fictícias sobre tecnologia, inovação e tendências futuras para demonstrar como um portal de notícias pode ser organizado visualmente.

## Objetivo do projeto

O principal objetivo deste projeto foi praticar conceitos de front-end através da recriação de um portal de notícias.

Durante o desenvolvimento, foram trabalhados conceitos como:

* Estruturação de páginas com HTML
* Estilização com CSS
* Responsividade
* Organização visual de conteúdo
* Uso de grids e flexbox
* Hierarquia de informações
* Reprodução de layouts inspirados em sites reais
